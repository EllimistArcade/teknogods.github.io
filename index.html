About

TeknoParrot is a software package allowing you to run selected PC-based arcade titles on your own hardware, with full support for keyboard and mouse controls, gamepads, steering wheels and joysticks.

We currently support games for various arcade systems, mapping the proprietary add-on hardware found in these systems to commodity PC peripherals, and fixing any other limitations in the game software that prevents them from running on standard Windows systems.

Setup

TeknoParrot

  1. Download latest TeknoParrot and extract archive somewhere.
  2. Open your extracted TeknoParrot folder and run TeknoParrotUi.exe.
  3. If this is your first time running TeknoParrot, you will be greeted with a message to set emulation settings. Click OK.
  4. If you have no games setup, you will be prompted to do so. You can say No for now.
  5. If you receive any popups to that updates are available, go ahead and click Update on each.
  6. At the top left of windows, click the hamburger icon, and choose Settings.
  7. sTo0z Zone is a controller's dead zone, most people feel comfortable around 20% or so. Adjust to your liking. If you need to setup your wheel axis settings, do so now. If you don't know, just try a game with your wheel and if see if you need to adjust.
  8. Click Save settings, and then Back.
  9. Click Hamburger menu again, and click Add Game.
  10. Choose the game from the list you have on your PC and wish to set up, then click ADD GAME.
  11. With your game selected from the Home menu, click GAME SETTINGS.
  12. Click Game Location box to choose where the appropriate game executable is. Specific game setups can be found on the Compatibility page of this site.
  13. Setup other options to your liking and click Save settings, then Back. Generally for an Xbox controller you want to select XInput, and everything else leave un-checked.
  14. Finally, click CONTROLLER SETUP.
  15. Click on each space for each button you want to set, then press the button you want on your controller.
  16. When finished, click Save settings, then Back.
  17. Repeat to setup any other games, when you want to play a game, select from the list and click LAUNCH GAME.
  18. Pressing ESC on your keyboard should exit the game.
  19. If you encouter issues, please look up a specific game in the Compatibility page of this site.

Common Issues / Questions I get an error extracting the archive! We have seen some users not able to extract the TeknoParrot archive if their software is too old. Please download latest update from software such as 7zip or WinRAR. My AntiVirus program reports a virus! Older versions of TeknoParrot project would trigger a false positive virus warning. TeknoParrot does not contain any malicious code. As of April 3, 2019, TeknoParrot has acquired an SSL certificate from COMODO, ensuring there is no virus associated with TeknoParrot. If you are still receiving a virus warning, please make sure you Download the latest version of TeknoParrot. How do I run on Mac or Linux? Currently this is a Windows only application. A game that is supported won't work! Please check the Compatibility page for specific game information. Why isn't this project open source? There are plans to fully open source this project, however the timeline for that is not set. What happens to Patreon funds? All support from Patreons goes toward development. This is to acquire hardware and other means of research. No funds are kept personally by any the developers. My issue isn't covered here! Please join the TeknoParrot Discord and bring your problem to the help channel. We have a lot of folks ready to help you out.
