SEGA nu (Vanilla/1.1/2/SX/SX 1.1)

About

Description:

Released in November 2013, nu is based on a mid-range PC running Windows Embedded 8 which got two revisions (1.1 and 2) in order to boost the rig. Later, "nu SX" and "nu SX 1.1", which are low-priced versions (equivalent to RingWide in the Ring generation), also appeared. All games thus far were only released in Japan, with the sole exception of Luigi's Mansion Arcade.

Specifications:

  • CPU (nu Vanilla/1.1): Intel Core i3-3220 3.30 GHz
  • CPU (nu 2): Intel Core i5-6500 3.20 GHz
  • RAM (nu Vanilla/1.1): 4 GB DDR3 SDRAM (PC3-12800)
  • RAM (nu 2): 8 GB DDR3 SDRAM (PC3-12800)
  • GPU (nu Vanilla): Nvidia GeForce GTX 650 Ti with 1 GB GDDR5 memory (Direct3D 11.1/OpenGL 4.3)
  • (nu 1.1): Nvidia GeForce GTX 750 Ti with 1 GB GDDR5 memory (Direct3D 11.1/OpenGL 4.4)
  • (nu 2): Nvidia GeForce GTX 950 with 1 GB GDDR5 memory (Direct3D 11.1/OpenGL 4.4)
  • Sound (nu Vanilla/1.1/2): 5.1 channel HD Audio
  • Output (nu Vanilla/1.1/2): VGC-P, UTT-P, twin display output
  • Connectors (nu Vanilla/1.1/2): JVS I/O connector, 4 channel serial port, 4× USB 3.0, 2× CAN bus
  • Networking (nu Vanilla/1.1/2): Gigabit Ethernet (1000BASE-T)
  • Storage (nu Vanilla/1.1/2): SATA SSD 64 GB, HDD 500 GB
  • OS (nu Vanilla/1.1/2): Microsoft Windows Embedded 8 Standard
  • (nu 2): Microsoft Windows Embedded 8.1 Standard
  • Media (Vanilla/1.1): Blu-ray or USB 3.0 storage, network delivery (SEGA ALL.NET)
