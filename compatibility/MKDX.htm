About

Game Title:
Mario Kart: Arcade GP DX

Hardware:
Namco ES3A

Game Info:
Mario Kart Arcade GP DX is a Mario Kart game for arcade systems, developed by Namco Bandai Games in partnership with Nintendo. It is the tenth installment in the Mario Kart franchise and the third title in the Arcade series following Mario Kart Arcade GP 2, which was a follow-up to Mario Kart Arcade GP. The game was released in Japan on July 25, 2013 and in the United States during 2014. Just like the previous two Arcade GP games, it is possible to do multiplayer play by linking up to four cabinets.

LAN Networking (2-? Players)

Setting up for LAN does not affect the ability to run this game in standalone mode.

Setup

  1. Edit the Game Settings within TeknoParrotUI.
  2. Set the "Gateway" network IP address to your "Default Gateway" address (Run "cmd.exe /k ipconfig")
  3. Run Game and enter the Test Menu.
  4. Enter Game Settings (second option down).
  5. Set a unique Cabinet ID and save.
  6. Exit Test Menu with test key.

Always allow Window Firewall to pass through communication of this game. This should auto pop-up upon first execution (Alt+Tab to view or exit game).

Known Working IP/Subnets

  • 192.168.0.x/255.255.255.0

Common Issues / Questions

Setup Banapass

Go to https://teknoparrot.com/OnlineProfile/MKDX (make sure you're logged in) and generate an ID number. Within TeknoParrotUI, input this ID number while setting up Mario Kart. In-game, press F2 to insert the card.

Graphic error during launch, or game launches in tiny window

Download missing files here.
Extract to respective folders inside Windows directory.

Camera image is rotated incorrectly

There is no in-game fix that you can do to resolve this, but there are specific cameras you can purchase that seem to rotate correctly for some reason. You can also physically rotate your camera.
