About

Game Title:
Wangan Midnight Maximum Tune 5

Hardware:
Namco ES3X

Game Info:
Wangan Midnight Maximum Tune 5 (also known as Maximum Tune 5 (named in the U.S.), or simply Maxi 5) is an arcade racing video game developed and published by Namco and based on the Wangan Midnight manga. It is the fifth game in the Wangan Midnight arcade game series. This is also the first game in the Wangan Midnight series to be officially relased in North America since Wangan Midnight Maximum Tune 3DX+.

Setup

Game

  • Executable is "wmnr5.exe" found in game's root folder.

LAN Networking (2-? Players)

Setting up for LAN affects the ability for the Slave cabinets to run this game in standalone mode. The Master cabinet can continue to operate as it provides the Terminal emulation for itself and all Slaves.

Setup

  1. Edit the Game Settings within TeknoParrotUI.
  2. Master (1 designated cabinet)
    • Enable "TerminalEmulator" setting.
    • Set the "NetworkAdapterIP" network IP address to 0.0.0.0
  3. Slave (all other cabinets)
    • Disable "TerminalEmulator" setting.
    • Set the "NetworkAdapterIP" network IP address to the "IPv4 Address" of the Master cabinet (Run "cmd.exe /k ipconfig" on Master)
  4. Run Game and enter the Test Menu.
  5. Enter Game Settings (second option down).
  6. Set a unique Cabinet ID and save.
  7. Exit Test Menu with test key.

Always allow Window Firewall to pass through communication of this game. This should auto pop-up upon first execution (Alt+Tab to view or exit game).

Known Working IP/Subnets

  • 192.168.0.x/255.255.255.0

Common Issues / Questions

I am stuck at "Connecting to Wangan Terminal"

Make sure TerminalEmulator is ticked, and that NetworkAdapterIP is set to 0.0.0.0 unless you're playing online. If you have too many network adapters installed on your PC, you may need to remove/disable them except for your primary network adapter.

Why is my car drifting to the side constantly?

Go into the TEST menu and calibrate your steering.

What is TerminalMode?

TerminalMode makes the game emulate the Terminal function rather then the racing cabinet function. With WMMT5, the cabinets all get their information like closing times and whether they're set to FreePlay or not from the Wangan Terminal cabinet, a standup machine that has a touchscreen that's used for customising cars on a Banapassport. Since we don't support Banapassport for WMMT5, TerminalMode has no use except for if you want to run the game in Event Mode with multiple machines, which can actually be emulated by our TerminalEmulator now.

What does the text in the yellow box mean?

It's just informing you that Banapassport cannot be used at this time.

Features Not Emulated

  • Full Banapass (saving)
  • Online Ghost Mode
  • Terminal Touchscreen
