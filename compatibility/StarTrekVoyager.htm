Setup

Game

  • Starting with 'voyager.chd'
  • Get 'chdman.exe' from a MAME release package
  • Open a cmd prompt, navigate to folder where 'chdman.exe' and 'voyager.chd' is
  • Use this command: chdman extracthd --input voyager.chd --output voyager.img
  • Wait for extract..."Extraction complete"
  • With a program like 7-Zip, open 'voyager.img' and extract '0.img', '2.img', and '3.img'
  • You can ignore '1.img' and '4'
  • Create a new folder somewhere and name it something like "Star Trek Voyager"
  • With a program like 7-Zip, right click '3.img' and extract 'stv' folder into your 'Star Trek Voyager' folder (you may need to run 7-Zip as administator)
  • If it yells at you about something, just choose "Yes to all", also ignore any extract warnings
  • With a program like 7-Zip, open '0.img' and navigate to 'home > rocky > stv'
  • Extract contents of 'stv' folder to your 'Star Trek Voyager' folder (you may need to run 7-Zip as administator)
  • With a program like 7-Zip, open '2.img' and open 'stv' folder
  • Extract contents of 'stv' folder to your 'Star Trek Voyager' folder (you may need to run 7-Zip as administator)
  • Game executable is 'trek.e4' found in your 'Star Trek Voyager' folder

Common Issues / Questions

Sound issues

Sound emulation is still missing music support, but don't doubt Harm's genius!

Game runs too fast

Game is made to run only 30fps, you must limit the fps with RTSS (or similar) or the game will run 2x speed.

Features Not Emulated

  • Music currently missing.
