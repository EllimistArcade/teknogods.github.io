About

Game Title:
Tokyo Cop

Hardware:
Gaelco PC Based (3D Hardware)

Game Info:
A driving game where the player assumes the role of a cop going after dangerous criminals in Tokyo from pickpockets to the most wanted throughout four districts: Ginza, Hibiya, Shinjuku and Shibuya. A promotion is earned based on how quickly criminals are captured. Different cars are available in each round. The game also has four selectable characters: Isumu, Kazu, Reika and Junko.

Setup

Game

  • Starting with 'tokyocop.chd'
  • Get 'chdman.exe' from a MAME release package
  • Open a cmd prompt, navigate to folder where 'chdman.exe' and 'tokyocop.chd' is
  • Use this command: chdman extracthd --input tokyocop.chd --output tokyocop.img
  • Wait for extract..."Extraction complete"
  • With a program like 7-Zip, open 'tokyocop.img' and extract '0.img'
  • You can ignore '1.img' and '2.img'
  • With a program like 7-Zip, right click '0.img' and choose "Extract here"
  • If it yells at you about something, just choose "Yes to all", also ignore any extract warnings
  • Open 'home > joc2001 > joc2001 > Sources > gameport > linux' folder
  • Copy 'gameport' to 'home > joc2001 > joc2001 > salidas' folder
  • The 'salidas' folder is the game folder you need, so rename it to 'Tokyo Cop' or so. copy this folder to wherever you keep your TeknoParrot games. You can safely delete everything else.
  • Game executable is 'gameport' now found in game's root folder ('salidas', which you hopefully renamed)
